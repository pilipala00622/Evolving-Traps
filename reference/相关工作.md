4. MemSkill: Learning and Evolving Memory Skills for Self-Evolving Agents[https://arxiv.org/pdf/2602.02474]:
目前大多数大模型 Agent 的记忆系统依赖于静态的、人工设计的操作。这篇论文提出了 MemSkill 框架，将记忆提取重构为可学习和自演化的“记忆技能”。它包含一个“控制器（Controller）”选择技能，一个“执行器（Executor）”生成记忆，以及一个“设计者（Designer）”会定期审查 Agent 交互中的困难案例，并自主演化出新的记忆处理技能。
5. Evo-Memory: Benchmarking LLM Agent Test-time Learning with Self-Evolving Memory [https://arxiv.org/abs/2511.20857]: 有评估大多关注静态问答，而忽视了经验复用。该论文提出了评估 LLM Agent 自演化记忆的框架和基准 Evo-Memory，同时提出了一种基于“行动-思考-记忆优化 (Action-Think-Memory refine)”的管道架构 ReMem。Agent 需要在连续任务流中实时检索、适应并“进化”其记忆。
6. MemGen: Weaving Generative Latent Memory for Self-Evolving Agents[https://arxiv.org/abs/2509.24704]:区别于传统外部检索式（Retrieval-based）记忆，该工作提出了生成式的隐式记忆框架 MemGen。实验表明，在没有显式监督的情况下，这种机制让智能体能自发演化出类似人类的记忆机能（如规划记忆、程序记忆和工作记忆）。
7. SAGE: Self-evolving Agents with Reflective and Memoryaugmented Abilities[https://arxiv.org/pdf/2409.00872]提出了具有反思与记忆增强能力的自演化多智能体框架。它引入了基于艾宾浩斯遗忘曲线的记忆优化机制，帮助 Agent 有选择地保留关键信息，并通过反馈和反思机制自主调整检索和规划策略。
8. 4D-ARE: Bridging the Attribution Gap in LLM Agent Requirements Engineering[https://arxiv.org/pdf/2601.04556]: 将 "Attribution-Driven"（归因驱动） 引入 LLM Agent 的设计中。文章指出 Agent 很容易产生“超出证据支持范围的因果幻觉（Hallucination beyond data）”，并提出了一种基于因果归因维度的多 Agent 设计规范，约束 Agent 给出带有强溯源证据的结论。

11. MemRL: Self-Evolving Agents via Runtime Reinforcement Learning on Episodic Memory[https://arxiv.org/pdf/2601.03192]: 提出 MemRL 框架，将冻结 LLM 的稳定推理与可塑的情景记忆解耦，通过基于情景记忆的运行时强化学习，实现 Agent 的自主演化，无需微调模型权重即可持续优化策略，同时避免灾难性遗忘
12. MA-RAG: From Conflict to Consensus: Boosting Medical Reasoning via Multi-Round Agentic RAG[https://arxiv.org/pdf/2603.03292]：提出多轮智能体 RAG 框架 MA-RAG，通过 Agent 迭代循环自主演化外部检索证据与内部推理历史，利用语义冲突信号主动识别知识缺口，迭代优化检索与生成逻辑，缓解推理过程中的幻觉累积
13. DRAG: Debate-Augmented RAG[https://aclanthology.org/2025.acl-long.770.pdf]-ACL 2025：提出辩论增强的 RAG 框架 DRAG，将多智能体辩论机制融入 RAG 的检索与生成全流程，通过支持者、反对者、裁判三类 Agent 的结构化对抗交互，提升检索证据的可靠性与生成内容的事实一致性，显著缓解 RAG 幻觉。
14. ReDeEP: Detecting Hallucination in Retrieval-Augmented Generation via Mechanistic Interpretability[https://arxiv.org/abs/2410.11414]-ICLR205: 提出 ReDeEP 框架，通过机制可解释性解耦 LLM 对检索知识与参数化知识的利用，精准定位 RAG 幻觉的来源，同时设计了 AARF 干预机制，无需微调即可动态优化模型的知识利用逻辑，缓解幻觉。
15. Identify, Isolate, and Purge: Mitigating Hallucinations in LVLMs via Self-Evolving Distillation[https://arxiv.org/html/2507.04680v1]-ACMMM2025-提出自演化蒸馏框架，通过模型自主识别幻觉、隔离错误信息、迭代净化生成内容的闭环，实现无需人工标注的模型持续优化，在多模态场景中显著缓解幻觉。
16. TOWARD FAITHFUL RETRIEVAL-AUGMENTED GENERATION WITH SPARSE AUTOENCODERS[https://arxiv.org/pdf/2512.08892]-ICLR2026: 提出 RAGLens 工具，通过稀疏自编码器实现 RAG 生成内容的 token 级归因解释，提供细粒度的幻觉检测反馈，同时基于该反馈实现了生成内容的迭代优化，显著降低 RAG 幻觉率。
17. Reflexion: Language Agents with Verbal Reinforcement Learning
核心是把反馈转成自然语言反思，写入 episodic memory buffer，后续决策时再读出来。
18. ExpeL: LLM Agents Are Experiential Learners（AAAI 2024）
它强调 Agent 从任务中积累经验、提炼自然语言知识、在新任务中检索复用，本质上是非参数更新的持续学习。
19. Self-RAG: Learning to Retrieve, Generate, and Critique through Self-Reflection
重点是让模型自己决定是否检索、如何 critique 当前输出，从而降低幻觉。
20. Corrective Retrieval Augmented Generation (CRAG)
偏 RAG 纠错机制。它会先评估检索结果质量，不行就触发修正或额外检索，属于“先发现检索问题，再纠错生成”的典型思路。
21. A-MEM: Agentic Memory for LLM Agents（NeurIPS 2025）
关注“Agent 记忆”。它不是简单向量存储，而是用 动态索引 + 链接式知识网络 去组织经验，和“记忆持续进化”高度相关。
22. MemGPT: Towards LLMs as Operating Systems
记忆系统视角非常强。把 LLM 当作操作系统，用分层记忆 / 虚拟上下文管理解决长程任务和长期对话中的遗忘问题。
23. Generative Agents: Interactive Simulacra of Human Behavior（UIST 2023）
核心结构就是 memory → reflection → planning，尤其适合你理解“反思如何从记忆中长出来”。
24. RAGTruth: A Hallucination Corpus for Developing Trustworthy Retrieval-Augmented Language Models
它给了一个专门针对 RAG 的幻觉语料和评测基准，适合做 detection / diagnosis / benchmark 支撑。
